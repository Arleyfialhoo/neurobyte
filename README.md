# NeuroByte 🧠💻

## 📌 Sobre o Projeto

**NeuroByte** é um projeto educacional criado para aproximar jovens da área de tecnologia. A iniciativa conta com um minicurso sobre desenvolvimento web e um quiz interativo. Assim, o projeto mostra que programação é acessível para todos — independentemente da idade ou experiência anterior.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Para a estruturação semântica das páginas.
* **CSS3:** Para a estilização visual, utilizando Flexbox, animações e efeitos.
* **JavaScript:** Responsável pela interatividade, lógica do quiz e comportamento dinâmico.
* **Google Fonts (Orbitron):** Fonte principal do projeto, reforçando a identidade visual tecnológica.
* **GitHub Pages:** Para o deploy e publicação do site.

O deploy foi realizado através do GitHub Pages, permitindo acesso direto pelo navegador.

## 📚 Principais Aprendizados

Durante a construção do NeuroByte, os seguintes conceitos foram praticados:

* Estruturação e estilização de interfaces web com HTML e CSS.
* Lógica de programação e manipulação do DOM com JavaScript.
* Organização de projetos em pastas e boas práticas de desenvolvimento.
